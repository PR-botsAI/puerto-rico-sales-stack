## 🇵🇷 Why Puerto Rico

The PR market has unique characteristics:

- **Bilingual reality** — Spanish-dominant with strong English overlay.
- **Relationship economy** — *confianza* (trust) is the currency before any close.
- **Family/community-centric decisions** — rarely a solo buyer; expect a *consejo familiar*.
- **High-context culture** — what is *not* said matters as much as what is.
- **Diverse industries** — auto, real estate, insurance, healthcare, retail, telecom, professional services, tourism.

This archive is filtered through that lens. Every framework includes notes on **how it translates** (or doesn't) to the PR buyer.
